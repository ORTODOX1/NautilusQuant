1. Случайное ортогональное вращение PolarQuant

Каждая пара координат вращается на случайный угол из PRNG. Делает координаты независимыми.
θ = random() × 2π
x' = x·cos(θ) − y·sin(θ)
y' = x·sin(θ) + y·cos(θ)

2. Перевод в полярные координаты PolarQuant

Пара (x, y) → (r, θ). Угловое распределение становится предсказуемым после вращения.
r = √(x² + y²)
θ = atan2(y, x)

3. Скалярное квантование Lloyd-Max Quantizer

Каждое значение сжимается до N бит. Нужны scale и zero-point для каждой группы → overhead.
levels = 2^bits
q = round((val − min) / (max − min) × (levels−1))
dequant = q / (levels−1) × (max − min) + min

4. QJL коррекция ошибок QJL

Остаточная ошибка → 1-битный знак. Устраняет bias скалярных произведений в attention.
error = original − quantized
sign_bit = sign(error) ∈ {−1, +1}
corrected = quantized + sign_bit × |error| × α

1. Ортогональное спиральное вращение NautilusQuant v2

Золотой угол 137.5° × индекс. БЕЗ масштабирования! Радиус сохранён: |v'| = |v|. Матрица строго ортогональна: T^T·T=I. Торнадо, не воронка.
θ = (2π / φ²) × (pairIndex + 1)
x' = x·cos(θ) − y·sin(θ)
y' = x·sin(θ) + y·cos(θ)
|v'| = |v| ✓   T^T·T = I ✓

2. Полярные координаты NautilusQuant

Аналогично TurboQuant, но распределение после спирали более предсказуемо → лучшее сжатие.
r = √(x² + y²)
θ = atan2(y, x)

3. Квантование (Fix 7: честное) NautilusQuant v2

Fix 7: Формула ИДЕНТИЧНА TurboQuant. Overhead одинаковый. Преимущество NautilusQuant — только в лучшем распределении углов (если оно лучше). Честное сравнение.
levels = 2^bits
q = round((val − min) / (max − min) × (levels−1))
overhead = 32 бит/группа (как у TurboQuant)

4. QJL коррекция NautilusQuant

Идентичен TurboQuant. 1 бит на значение. Универсален.
